📌 Project: QR Code Generator
🔹 Description

A simple QR Code Generator that takes any URL as input and generates a corresponding QR code image instantly.

⚙️ Tech Stack
Node.js
JavaScript

📦 Used Modules / Imports
fs (File System module)
qr-image (for generating QR codes)

🚀 Features
Convert any URL into a QR code
Saves QR code as an image file
Fast and lightweight
Easy to use

🛠️ Working
User provides a URL
QR code is generated using the qr-image module
The image is saved locally using fs

📂 Output
Generates a .png QR code image for the given link
